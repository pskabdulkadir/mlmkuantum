# AKN Group MLM System

A comprehensive multi-level marketing (MLM) system integrated with spiritual features, banking, and product management.

## Features

- **Monoline MLM System**: Dynamic 7-level single-line architecture.
- **E-wallet & Financials**: Integrated wallet for commissions, withdrawals, and transfers.
- **Spiritual Panel**: Features like Quran pages, hatim tracking, and dua requests.
- **Comprehensive Admin Panel**: Full control over users, products, commissions, and system settings.
- **Clone Pages**: Personalized landing pages for members to recruit new users.
- **Training System**: Integrated Zoom training and notifications.

## Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Motion.
- **Backend**: Node.js, Express, MongoDB/Mongoose.
- **Persistence**: MongoDB for core data, Redis (optional) for queues.

## Quick Start

1. Configure `.env.example` to `.env`.
2. Run `npm install`.
3. Start development server: `npm run dev`.
